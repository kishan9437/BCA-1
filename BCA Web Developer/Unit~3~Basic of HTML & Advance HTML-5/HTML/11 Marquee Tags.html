Marquee Tags:

Scroll text,graphics,images...inside a web page using marquee element.
attribute:-
-behavior=scroll,slide,alternate
-directoin=left,right,up,down
-bgcolor=background color(colour name)
-height=pixel or percentage
-width=pixel or percentage
-scrollamount=increase a speed of marquee
-scrolldelay=decrease a speed of marquee
